# BIGC CTF Writeups

CTF 竞赛解题记录与学习笔记，使用 MkDocs Material 构建。

在线站点：<https://starnotes-xj.github.io/BIGC_CTF_Writeups/>

## 内容概览

- `CTF_Writeups/`：文档源目录（Writeups 与站点页面）
  - `Web/`、`Cryptography(密码学)/`、`Reverse Engineering(逆向工程)/`、`Misc(杂项)/`、`PPC/`
  - `events/`：按比赛归档
  - `tags.md`、`WRITEUP_TEMPLATE.md`
- `scripts_go/`、`scripts_python/`：辅助脚本
- `files/`：题目附件与相关文件
- `overrides/`、`stylesheets/`、`javascripts/`：站点自定义资源
- `mkdocs.yml`：站点配置
- `site/`：构建输出目录

## 本地预览

```bash
mkdocs serve
```

> 需要本地已安装 MkDocs Material。
